Roo Code é uma extensão do VS Code que traz assistência de programação com IA diretamente no seu editor. Instale usando um destes métodos:
1. **VS Code Marketplace (Recomendado)** - método mais rápido para usuários padrão do VS Code e Cursor
2. **Open VSX Registry** - para editores compatíveis com VS Code como VSCodium
---

## VS Code Marketplace

1. Abra o VS Code
2. Acesse Extensões: Clique no ícone de Extensões na Barra de Atividades ou pressione `Ctrl+Shift+X` (Windows/Linux) ou `Cmd+Shift+X` (macOS)
3. Pesquise por "Roo Code"
4. Selecione "Roo Code" por RooVeterinaryInc e clique em **Instalar**
5. Recarregue o VS Code se solicitado

Após a instalação, encontre o ícone do Roo Code (<KangarooIcon />) na Barra de Atividades para abrir o painel do Roo Code.

## Open VSX Registry

Para editores compatíveis com VS Code sem acesso ao Marketplace (como VSCodium e Windsurf):

1. Abra seu editor
2. Acesse a visualização de Extensões
3. Pesquise por "Roo Code"
4. Selecione "Roo Code" por RooVeterinaryInc e clique em **Instalar**
5. Recarregue se solicitado

## Instalação Manual via VSIX

Se preferir baixar e instalar o arquivo VSIX diretamente:

1. **Baixe o arquivo VSIX:**
   * Encontre versões oficiais na [página de Releases do Roo Code no GitHub](https://github.com/RooCodeInc/Roo-Code/releases)
   * Baixe o arquivo `.vsix` do release mais recente

2. **Instale no VS Code:**
   * Abra o VS Code
   * Acesse a visualização de Extensões
   * Clique no menu "..." na visualização de Extensões
   * Selecione "Instalar a partir de VSIX..."
   * Navegue e selecione seu arquivo `.vsix` baixado

## Solução de Problemas

<img src="/img/installing/installing-4.png" alt="Painel Output do VS Code mostrando logs do Roo Code para solução de problemas" width="100%" />
*Painel Output do VS Code mostrando logs do Roo Code para solução de problemas*

**Extensão Não Visível**
* Reinicie o VS Code
* Verifique se o Roo Code está listado e habilitado em Extensões
* Tente desabilitar e habilitar novamente
* Verifique o painel Output por erros (Visualizar → Output, selecione "Roo Code")

**Problemas de Instalação**
* Garanta uma conexão estável com a internet
* Verifique a versão do VS Code 1.84.0 ou superior
* Se o VS Code Marketplace estiver inacessível, tente o método Open VSX Registry
